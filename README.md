# AgentStack

AgentStack is a lightweight framework for building AI agents with tools, memory, and reasoning loops.
The goal of AgentStack is to provide a simple and transparent framework for developers who want to build and experiment with agent-based systems.

---

## Features

- **Simple agent reasoning loop**: Efficient decision-making cycles for autonomous tasks.
- **Tool integration**: Easily connect agents to external APIs and functions.
- **Claude model support**: Optimized for Anthropic's Claude series.
- **Modular architecture**: Swap out models, memory types, or tools with ease.
- **CLI interface**: Streamlined interaction directly from your terminal.

---

## Installation

Clone the repository:
```bash
git clone https://github.com/Adwaith-R-Nair/AgentStack.git
cd AgentStack
```

Install dependencies:
```bash
pip install -r requirements.txt
```
