# Contributing to AgentStack

Thank you for your interest in contributing to **AgentStack**.

AgentStack is an open-source framework designed to help developers build AI agents easily. Contributions from the community help improve the framework and expand its capabilities.

---

# How to Contribute

You can contribute in several ways:

* reporting bugs
* fixing issues
* improving documentation
* creating new tools
* adding model integrations
* writing example agents

---

# Development Setup

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/agentstack.git
cd agentstack
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

# Project Structure

The repository follows a modular structure.

```
agentstack/
  core/
  models/
  tools/
  memory/
  utils/

examples/
docs/
tests/
```

Each module has a specific responsibility.

---

# Contribution Workflow

1. Fork the repository.
2. Create a new branch.

Example:

```bash
git checkout -b feature/add-new-tool
```

3. Implement your changes.

4. Commit your changes.

Example:

```bash
git commit -m "feat: add weather tool"
```

5. Push your branch.

```bash
git push origin feature/add-new-tool
```

6. Open a Pull Request.

---

# Coding Guidelines

To maintain code quality:

* Write clear and readable code
* Follow existing project structure
* Keep functions small and modular
* Add comments when necessary

---

# Reporting Bugs

If you encounter a bug:

1. Open a GitHub issue
2. Describe the bug clearly
3. Include reproduction steps
4. Provide relevant logs

You may also add the bug to:

```
docs/bug-tracker.md
```

---

# Feature Requests

Feature suggestions are welcome.

When proposing a feature:

* explain the use case
* describe the expected behavior
* provide examples if possible

---

# Documentation Contributions

Documentation improvements are extremely valuable.

You can help by:

* improving guides
* fixing typos
* adding examples
* explaining architecture

---

# Community Guidelines

Please follow these principles:

* be respectful
* encourage collaboration
* provide constructive feedback
